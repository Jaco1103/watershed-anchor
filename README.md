# Watershed × Viah - working session anchor

A linear, present-from-the-top HTML deck used to anchor a 1-hour session.

**Part 1 - Watershed:** what it is, how the operator chain works, a zoom into the `archetype` operator, and the core ask - what 3-4 questions a data architect would force before any database decision.

**Part 2 - Viah:** the real Supabase/Postgres + PostGIS design - data model, the ingest / enrich-once / geo pipeline, and the deterministic `search_places` scoring (tag match + proximity + rating) - then where it strains at scale.

Open `index.html` (or the Pages link). Arrow keys / on-screen arrows to move through it.
